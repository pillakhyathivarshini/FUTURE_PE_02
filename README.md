# FUTURE_PE_02
# AI-Powered UGC Ad Content Pack

## Project Overview

This project was created as part of the Prompt Engineering Internship by Future Interns.

The goal of this project is to use AI tools like ChatGPT to create high-converting UGC-style advertisement content for a premium café brand.

## Business Chosen

**Café Serene** — a cozy and elegant café in Bangalore serving students, families, young professionals, and coffee lovers seeking a relaxing café experience.

## Features

* UGC-style ad hooks
* Instagram Reel scripts
* CTA-focused content
* AI-generated captions
* Reusable prompts
* YouTube Shorts ad scripts
* Facebook ad copy

## Tools Used

* ChatGPT
* GitHub
* Lovable AI (for website creation and branding)

## Project Structure

* `scripts/` → Ad scripts
* `hooks.txt` → Scroll-stopping hooks
* `captions.txt` → Social media captions
* `prompts.md` → Prompts used
* `README.md` → Project overview

## Learning Outcomes

* Improved prompt engineering skills
* Learned UGC ad content creation
* Understood AI-based marketing workflows
* Practiced creating platform-specific advertisement scripts
* Developed brand-focused promotional content using AI

## Platforms Covered

* Instagram Reels
* Facebook Ads
* YouTube Shorts

## Outcome

This project helped in understanding how prompt engineering and AI tools can be used to create engaging UGC-style marketing content for local businesses like Café Serene. The project demonstrates how AI can make social media promotion more creative, efficient, and impactful while maintaining a premium café brand identity.
